
# ATM Simulator

> A command-line banking system built to strengthen core C++ programming fundamentals.

# Overview

ATM Simulator is a menu-driven C++ application that replicates the core workflow of a banking ATM. It implements PIN authentication, balance management, deposits, withdrawals, and transaction validation through a clean console interface.

The project focuses on writing structured, reliable program logic while reinforcing procedural programming concepts.

---

# Features

* PIN authentication
* Balance inquiry
* Deposit funds
* Withdraw funds
* Transaction validation
* Overdraft protection
* Menu-driven CLI
