
# BIOS Portfolio

> A portfolio designed as firmware, not a website.

# Overview

BIOS Portfolio reimagines the traditional developer portfolio as an interactive BIOS interface. Inspired by classic PC firmware, it replaces conventional navigation with a keyboard-driven experience featuring a simulated boot process, system menus, and a retro visual design.

Built with Django and vanilla web technologies, the project focuses on creating an immersive interface while maintaining a clean, modular architecture.

---

# Features

* BIOS-inspired interface
* Simulated boot sequence
* Keyboard-first navigation
* Interactive project explorer
* Resume viewer
* Responsive layout
* Modular architecture
